Highlights

This is a toolbox deck using the Tzimisce and their VAI disciplines.

The combat module uses {{ card("Chiropteran Marauder") }} for aggravated damage, Intercept is high with 27 reactions combining A and I. Political actions are used to deliver pool damage with {{ card("Kine Resources Contested") }} while {{ card("Changeling") }} and {{ card("Mind of the Wilds") }} are used for stealth.

{{ card("Blood Doll") }} and {{ card("Villein") }} are used to get some pool back.

The deck is a bit slow but very difficult to predict for opponents.

Tips & Tricks

This version of the deck includes a lot of cards in only 1 copy. This is a bit rare, but it makes the deck even harder to figure out for opponents. One should be careful to keep the opponents guessing as to what the deck can really do.

{{ card("Under Siege") }} is a powerful deterrent against the predator: knowing wakes are available is far more troublesome than suspecting one has it in had.

Variants

There is a lot of room for variations, including in the crypt: {{ card("Sascha Vykos, The Angel of Caine") }}, {{ card("Stravinsky") }} or {{ card("Meshenka") }} can get a star position with 3 copies.

A bit more permanent intercept is a common inclusion, with {{ card("Bowl of Convergence") }}, {{ card("Revenant") }} and {{ card("Mr. Winthrop") }}. Permanent bleed enhancement is often included too, be it with {{ card("Pulse of the Canaille") }}, {{ card("J. S. Simmons, Esq.") }} and {{ card("Tasha Morgan") }}, or {{ card("Living Manse") }}.
