Rachel Madness

{% endblock %} {% block archetype %}

Highlights

This is a stealth & bleed deck starring {{ card("Rachel Brandywine") }}. The idea is to use her J discipline to fetch an {{ card("Homunculus") }} which, alongside {{ card("Madness Network") }}, will let her bleed during multiple consecutive turns.

Stealth is high, with 15 modifiers and {{ card("Form of Mist") }}.

On the defence side, {{ card("Second Tradition: Domain") }} and {{ card("Eyes of Argus") }} are used to intercept and protect the {{ card("Madness Network") }}, with many copies of {{ card("Earth Meld") }} to avoid combat and stay unlocked.

The deck is slow in the beginning but has a decent defence, and once the combination hits the table, it can finish a prey very quickly.

Tips & Tricks

{{ card("Enkil Cog") }} is a cheap way to have {{ card("Rachel Brandywine") }} bleed for 3 instead of 2. Moreover, it can replace {{ card("Madness Network") }} a bit if it is destroyed.

{{ card("Homunculus") }} costs blood, so {{ card("Perfectionist") }}, {{ card("Giant's Blood") }} and {{ card("The Rack") }} are essential to feeding the combination.

{{ card("Greger Anderssen") }} is an excellent sidekick as he also has j and can therefore use an {{ card("Homunculus") }} too. Moreover, he is also Prince, meaning he gets to play {{ card("Parity Shift") }} and {{ card("Second Tradition: Domain") }} too. If {{ card("Protean") }} is drawn, he can even use {{ card("Earth Meld") }} at its full potential.

{{ card("Blessing of Chaos") }} is a useful addition, as it can cripple bleed modules a bit, forcing them to use their bleed modifiers before blocks are announced.
