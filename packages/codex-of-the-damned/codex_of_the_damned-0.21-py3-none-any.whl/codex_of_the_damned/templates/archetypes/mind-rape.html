Highlights

This is a special deck using {{ card("Mind Rape") }} alongside a big capacity vampire with DO, {{ card("Lucian") }} here. The idea is to steal the opponents vampires and use {{ card("Heidelberg Castle, Germany") }} to deprive them of their blood and equipment.

Using Malkavian and {{ card("Madness Network") }} brings an edge to this idea, as {{ card("Mind Rape") }} can be played on the predator turn for example, to get control of the vampire in the immediately following turn.

The deck uses {{ card("Second Tradition: Domain") }} for defence, together with {{ card("Obedience") }} and {{ card("Mental Maze") }} to avoid combat.

Bleed and votes are used to apply pool damage, in combination with a strong stealth module using O.

Tips & Tricks

{{ card("Scourge of the Enochians") }} is used as a counter against breed constructs that can be overwhelming for this deck.

{{ card("Metro Underground") }} lets a vampire bleed or vote, then unlock at the end of the turn, only to use {{ card("Madness Network") }} for another action during another Methusalah's turn.

Variants

There is a Ventrue variant using {{ card("Arika") }} as a star. {{ link("/deck-search", _("Erol Hammer's deck"), twda_id="2k9fcqnac") }} from 2009 is a good example.
